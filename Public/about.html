Project Description

This project demonstrates how a web application can be automatically deployed using CI/CD pipelines.

Technologies Used

  • HTML5
  • CSS3
  • Node.js
  • Express.js
  • GitHub Actions
  • Azure App Service

Project Objectives

  • Create a simple web application.
  • Host the application on Azure.
  • Implement CI/CD using GitHub Actions.
  • Automate deployments after code changes.
